# Silan Desktop Pet (丝蓝桌宠)

基于 PyQt6 和 Quartz 开发的 macOS 桌面宠物，角色形象为丝蓝。

## 功能特点
*   **呼吸感动画**：模拟自然的缩放呼吸效果。
*   **智能吸附**：自动外挂于当前活动窗口的右侧边缘（非重叠模式）。
*   **始终置顶**：宠物窗口始终保持在屏幕最前端。

## 环境要求
*   Python 3.9+
*   PyQt6
*   pyobjc-framework-Quartz
